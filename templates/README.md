# Templates

This folder contains the framework references and AI prompts that power
Drumbeat's launch workflow.

## Important: copyright

The original templates from the Product Marketing Alliance course are
copyrighted by PMA. **Do not commit verbatim PMA template text to this
public repo.** What is allowed:

- ✅ Framework *names* and high-level descriptions (factual references)
- ✅ Original AI prompts that *apply* the methodology in our own words
- ✅ Our own derivative templates inspired by the frameworks
- ❌ Verbatim text from PMA materials, slide decks, or worksheets

If you need a working file with verbatim PMA content for personal
reference, put it under `templates/private/` (gitignored) — never commit it.
